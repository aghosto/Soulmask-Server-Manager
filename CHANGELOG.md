# 1.0.0
- 正式版发布

# 1.0.1
- 增加 '服务器导入' 功能，现在可以直接把指定文件夹拖到弹出窗口中实现导入服务器（导入需等待一段时间）
- 增加 '定时重启服务器' 功能
- 优化部分交互界面（log界面字体等）
- 增加 'Rcon远控' 功能，但并未对灵魂面甲做针对性配置
- 增加 '验证更新' 选项，在更新服务器的时候可以选择是否验证（validate），但一般建议是打开的

# 1.0.2
- 修复多个mod只生效第一个的问题

# 1.0.3
- 修复了更新游戏服务器时状态异常的问题

# 1.0.4
- 增加 'Rcon远控' 端口和密码配置行，现在可以配置每个服务器的Rcon端口和密码
- 修复 'Rcon远控' 功能，并对灵魂面甲的指令做针对性配置优化（具体功能还需自己学习使用）
- 修改了部分UI交互界面布局
