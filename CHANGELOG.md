# 1.0.0
- 正式版发布

# 1.0.1
- 增加 `服务器导入` 功能，现在可以直接把指定文件夹拖到弹出窗口中实现导入服务器（导入需等待一段时间）
- 增加 `定时重启服务器` 功能
- 优化部分交互界面（log界面字体等）
- 增加 `Rcon远控` 功能，但并未对灵魂面甲做针对性配置
- 增加 `验证更新` 选项，在更新服务器的时候可以选择是否验证（validate），但一般建议是打开的

# 1.0.2
- 修复多个mod只生效第一个的问题

# 1.0.3
- 修复了更新游戏服务器时状态异常的问题

# 1.0.4
- 增加 `Rcon远控` 端口和密码配置行，现在可以配置每个服务器的Rcon端口和密码
- 修复了 `Rcon远控` 功能，并对灵魂面甲的指令做针对性配置优化（具体功能还需自己学习使用）
- 修改了部分UI交互界面布局

# 1.0.5
- 修复了导入服务器时主窗口卡顿的问题
- 修复了启动服务器时软件控制台不显示启动成功的问题

# 1.1.0
- 增加 `服务器状态` 数据栏，可定时刷新服务器内玩家在线情况（使用Telnet，注意修改各服务器的监听端口）
- 增加 `打开Logs文件夹` 按键，可快速打开log文件夹以反馈用

# 1.2.0
- 增加 `管理模组` 功能，点击按钮打开窗口即可对对应服务器的Mod进行添加、取消订阅以及查看Mod信息等功能，打开窗口自动解析steam创意工坊的mod（需电脑能上创意工坊）
- 左上角可直接输入Mod名称或者id进行搜索（作者名称不可以）
- 对着指定的模组右键弹出菜单，可选择复制对应属性或者只看单个作者的Mod列表
- 在 `订阅` 一列中将需要的Mod勾选，然后点击上方 `保存Mod订阅信息` 即可保存mod订阅 ***注意：如果mod在当页没有的情况下先保存单页已经勾选了的，再翻页勾选保存别的页数的Mod***
- 若想取消订阅Mod，*需要找到对应的Mod取消勾选订阅，然后点击保存即可取消订阅* ，如果比较难找，那就在 `服务器已订阅Mods` 数据栏中右键对应Mod获取Id到上方搜索框搜索后取消订阅即可

# 1.2.1
- 增加 `导入地图存档` 功能，可以把服务器 **根目录** 或者 **地图存档文件(world.db)** 直接拖到弹出窗口中
- 修改部分UI布局
